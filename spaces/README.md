# Visatra

Real-time object detection using YOLOv8 + MediaPipe Hand Gesture Recognition.

## Features

- Image upload → YOLO object detection
- Hand landmark visualization  
- Peace gesture / Open palm detection

## Tech Stack

- Flask (web server)
- YOLOv8 (Ultralytics)
- MediaPipe Hand Landmarker
- OpenCV
